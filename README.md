# 🤖 AI Resume Screening & Job Match System

An end-to-end NLP + ML system that screens resumes, extracts skills, and matches candidates to the best-fit jobs — with a live Streamlit dashboard.

---

## 🚀 Live Demo

```bash
streamlit run app.py
```

Then open http://localhost:8501 in your browser.

## ⚙️ Features

| Feature | Description |
|---|---|
| 📄 PDF Upload | Extract text from any resume PDF |
| 🛠️ Skill Extraction | Detects 80+ technical skills via NLP |
| 🧠 Semantic Matching | Sentence Transformers / TF-IDF cosine similarity |
| 🎯 Match Scoring | Weighted composite score (skill + semantic + experience) |
| 💡 Skill Gap Analysis | Shows missing skills with learning resource links |
| 📊 Visualizations | Bar charts, radar charts, progress bars |
| 📥 CSV Export | Download your match results |

---

## 🧮 Scoring Formula

```
Final Score = 0.50 × Skill Match Score
            + 0.35 × Semantic Similarity Score
            + 0.15 × Experience Score
```

## 🛠️ Tech Stack

- **Python 3.10+**
- **Streamlit** — Web UI
- **spaCy** — NLP / Named Entity Recognition
- **sentence-transformers** — Semantic similarity (all-MiniLM-L6-v2)
- **scikit-learn** — TF-IDF, cosine similarity, evaluation metrics
- **pdfplumber** — PDF text extraction
- **Plotly** — Interactive charts
- **Pandas / NumPy** — Data processing

## 📦 Installation

```bash
# 1. Clone the repo
git clone https://github.com/yourusername/ARS-JMS.git
cd ARS-JMS

# 2. Create virtual environment
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # Mac/Linux

# 3. Install dependencies
pip install -r requirements.txt

# 4. Download spaCy model
python -m spacy download en_core_web_sm

# 5. Run the app
streamlit run app.py
```

## 📊 Dataset

- Data Science / ML / AI
